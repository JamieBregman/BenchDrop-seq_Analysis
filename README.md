# BenchDrop-seq_Analysis

Scripts for processing and visualization of BenchDrop-seq data used for the analyses in "BenchDrop-seq: a microfluidics-free platform for benchtop single-cell long-read RNA sequencing." A pre-print version of this work can be found on BioRxiv.

Raw and processed BenchDrop-seq single-cell short-read and long-read sequencing data are avaialble through the NIH Gene Expression Omnibus (GEO). Accession ID: GSE318099.
